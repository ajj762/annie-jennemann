Missouri Bills Database Project

During the spring of 2022, I worked on a four-month long reporting project on data, graphics and other visual storytelling. This reporting project was a multi-story and multiplatform project about the changing landscape of Missouri's rural communities.

During this project, we wanted to track the decisions made that led to inequities within these communities on a statewide level. We found there was no good way to search issues by keyword for multiple years.

I put together a database of Missouri bills truly agreed and finally passed from 2000 to 2022 that is searchable. This project was for our own use, but is publicly available for anyone to replicate and use. Below is the database. Visit the GitHub repository or the main URL for the database.
